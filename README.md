## 快速开始（30秒上手）

```bash
# 1. 克隆项目
git clone https://github.com/你的用户名/你的仓库名.git
cd 你的仓库名

# 2. 安装依赖（根据你的项目修改）
npm install          # 前端 / Node 项目
# 或
pip install -r requirements.txt   # Python 项目
# 或
cargo build --release             # Rust 项目

# 3. 运行
npm run dev          # 开发模式
# 或
python main.py
# 或
./target/release/你的可执行文件
